INSTRUÇÕES PARA CRIAR O REPOSITÓRIO NO GITHUB:

1. **Acesse seu GitHub** em: https://github.com

2. **Clique no botão "+"** (canto superior direito) → "New repository"

3. **Preencha os campos:**
   - Repository name: onde-parei-app
   - Description: App Flutter para gerenciar leitura de mangás e livros
   - ⚠️ Visibility: PRIVATE (MARCADO)
   - ❌ DESMARQUE "Add a README file"
   - ❌ DESMARQUE "Add .gitignore"
   - ❌ DESMARQUE "Choose a license"

4. **Clique em "Create repository"**

5. **Na próxima tela, copie o comando de push:**

   No VS Code, execute estes comandos:

   git remote add origin https://github.com/SEU_USERNAME/onde-parei-app.git

   git branch -M main

   git push -u origin main

🔐 **IMPORTANTE - SEGURANÇA:**
- Mantenha este repositório PRIVADO
- Não committe chaves da API públicas
- Configure o Firebase apenas localmente (não no Git)

⚙️ **Para configurar o Firebase no seu PC:**
Execute: flutterfire configure

Após a configuração, seu firebase_options.dart será criado localmente (não vai para o Git).
